
# Portofolio Guru Nasional

Website portofolio pendidikan milik **Eka Amin Hidayat**.

## Fitur
- Hero slider
- Profil guru
- Timeline karir
- Novel digital
- Galeri kegiatan
- Lightbox foto

## Cara Upload ke GitHub

1. Buat repository baru di GitHub
2. Upload semua file dalam folder ini
3. Masuk ke Settings → Pages
4. Pilih Branch: main dan Folder: /root
5. Website akan online otomatis

## Struktur

index.html  
ebook.html  
/img (isi dengan foto)  
/music (isi dengan musik)
